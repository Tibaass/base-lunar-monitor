# Monitoramento de Recursos da Base Lunar

Trabalho da disciplina Advanced Programming and Mobile Dev (FIAP - 2EMR).

App em React Native que consome uma API Spring Boot pra monitorar recursos de uma base lunar (agua, energia, climatizacao, oxigenio).

## Como rodar

**Backend** (JDK 17):
```bash
cd backend
mvnw.cmd spring-boot:run
```

**App** (Node):
```bash
cd mobile
npm install
npm run web
```

## Tecnologias

- Backend: Java 17, Spring Boot, H2
- Mobile: React Native (Expo), TypeScript, axios
